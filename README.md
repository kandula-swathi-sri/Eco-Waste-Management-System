# Eco-Waste Management System

A gamified web application developed using Flask that encourages sustainable waste management through interactive features such as waste scanning, eco-challenges, rewards, and a points-based system.

## Features

- ♻️ Waste Scanner
- 🎯 Sustainability Challenges
- 🏆 Points and Rewards System
- 🎡 Lucky Wheel
- 📊 User Dashboard
- 🌱 Eco-friendly Task Tracking

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/kandula-swathi-sri/Eco-Waste-Management-System.git
```

2. Navigate to the project

```bash
cd Eco-Waste-Management-System
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
